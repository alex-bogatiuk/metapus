# Горячие клавиши (Keyboard Shortcuts)

> **TL;DR:** Система использует паттерн централизованного реестра (`Zustand`) и один глобальный обработчик (Listener). Добавление шортката сводится к одному вызову хука `useShortcut` в любом компоненте.

> **Тип:** Reference
> **Аудитория:** Frontend Developer

---

## 1. Регистрация шортката

Чтобы добавить шорткат в компонент, используйте хук:

```typescript
import { useShortcut } from "@/hooks/useShortcut"

function MyForm() {
    const handleSave = () => { /* ... */ }

    useShortcut(
        "editing.save",    // Уникальный ID
        "mod+s",           // Кроссплатформенная комбинация
        "Записать",        // Label для меню помощи
        "editing",         // Группа (editing, list, navigation, general)
        handleSave         // Callback
    )
}
```

**Что происходит под капотом:**
1. При маунте компонента шорткат добавляется в `useShortcutStore`.
2. При анмаунте — удаляется.
3. Шорткат автоматически появляется в модальном окне Help Dialog (`Ctrl+/`).
4. Если фокус находится внутри `<input>` или `<textarea>`, одинарные клавиши (без Ctrl/Alt) автоматически подавляются (Input Suppression).

## 2. Кроссплатформенность (`mod`)

Используйте спецификатор `mod` вместо жесткой привязки к `Ctrl` или `Meta` (`Cmd`).
- **Windows/Linux:** `mod+s` интерпретируется как `Ctrl+S`.
- **macOS:** `mod+s` интерпретируется как `Cmd+S` (`⌘S`).

**Обработка раскладки:** Библиотека работает с физическими клавишами (`event.code`). Поэтому `mod+s` будет работать корректно даже если у пользователя включена русская раскладка клавиатуры (где на клавише `S` находится буква `Ы`).

## 3. Решение конфликтов

Если разные компоненты на экране пытаются зарегистрировать один и тот же шорткат (например, глобальный `Ctrl+S` и локальный `Ctrl+S` формы), можно использовать параметр `priority`:

```typescript
useShortcut("global.save", "mod+s", "Сохранить всё", "general", globalSave, { priority: 0 })
useShortcut("form.save", "mod+s", "Сохранить документ", "editing", formSave, { priority: 10 })
```
При нажатии `Ctrl+S` сработает тот шорткат, у которого `priority` выше. Когда форма закроется, её шорткат удалится, и приоритет вернется к глобальному обработчику.
